# 系统安全扫描

对操作系统进行安全检查，发现潜在风险。

## 执行步骤

### 1. 网络安全
- 检查当前监听的端口，识别异常端口和未知进程
- 检查防火墙状态
- 检查 hosts 文件是否被篡改

### 2. 用户与权限
- 列出系统用户账户，检查是否有可疑账户
- 检查管理员组成员
- 检查是否有异常的计划任务

### 3. 启动项与服务
- 检查开机启动项
- 检查可疑的系统服务
- 检查最近安装的程序

### 4. 敏感文件暴露
- 检查桌面、下载目录是否有暴露的密钥/密码文件（.pem, .key, .env, credentials 等）
- 检查 .ssh 目录权限和已有密钥

### 5. 系统更新
- 检查系统更新状态，是否有未安装的安全补丁

## 输出格式

按严重程度分级：
- **严重**：必须立即处理
- **警告**：建议处理
- **建议**：可以改进

对每个问题给出具体的修复命令或操作步骤。
